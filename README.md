##  使用

前往[release](https://github.com/ravizhan/wyk8_crack/releases)下载最新版本的软件

**解压后**点击`main.exe`运行软件，按照提示操作即可。

也可直接下载源码运行
```commandline
git clone https://github.com/ravizhan/wyk8_crack.git
cd wyk8_crack
pip install -r requirements.txt
python main.py
```
